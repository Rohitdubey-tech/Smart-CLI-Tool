# 🚀 Smart CLI Tool

An all-in-one **Python Command Line Application** designed to revise and demonstrate core Python concepts through real-world features.

---

## 📌 Features

* 🧮 **Calculator**
  Perform basic arithmetic operations (+, -, *, /)

* 🔐 **Password Generator**
  Generate secure random passwords

* 📝 **To-Do List Manager**
  Add, view, and delete tasks (stored in JSON)

* 💰 **Expense Tracker**
  Track and calculate your daily expenses

---

## 🧠 Concepts Covered

This project is built to revise **complete Python fundamentals**:

* Functions & Modular Programming
* File Handling (JSON)
* Exception Handling
* Lists, Dictionaries
* CLI Interaction (`input`, `print`)
* Code Structuring

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Rohitdubey-tech/Smart-Tool.git
```

2. Navigate to the folder:

```bash
cd SmartTool
```

3. Run the project:

```bash
python3 main.py
```
